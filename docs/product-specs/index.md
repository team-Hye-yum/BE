# Product Specs Index

제품 스펙은 구현 지시가 아니라 검증 가능한 수용 기준입니다.

## 스펙에 포함할 것

- 사용자와 권한
- 성공 시나리오
- 실패 시나리오
- API 요청/응답
- 오류 코드와 HTTP 상태
- 트랜잭션 기대
- 로그와 관측 포인트
- 테스트 범위

## DIVE 2026 주요 사용자

- 부산TP 사업 담당자
- 외부 평가위원
- 신청기업 담당자 PoC 사용자

## DIVE 2026 주요 화면

- 기업 검색/선택
- 기업 종합 스코어카드
- 지원이력 타임라인
- 검토신호 상세
- 지원 이후 변화 확인
- 평가 근거 요약 카드
- 신청 전 기업정보 점검
